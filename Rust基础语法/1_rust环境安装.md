# 1 安装Rust
- 官方网址:  https://www.rust-lang.org 进入后点击install进入安装界面。
- 根据界面提示下载安装即可，如果是linux系统，会给一个终端命令，使用终端命令来安装
    ```shell
    curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
    ```
- 查看rust版本
    ```shell
    rustc --version
    ```
- 更新Rust
    ```shell
    rustup update
    ```
- 卸载rust
    ```shell
    rustup self uninstall
    ```
- 在浏览器打开本地rust文档
    安装rust的时候还会下载rust文档
    ```shell
    rustup doc
    ```
