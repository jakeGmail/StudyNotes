# 1. GenRandomString
<font color=green>**方法签名**</font>： `GenRandomString(size uint8) string`

<font color=green>**功能**</font>： 生成指定长度的随机字符串

<font color=green>**参数**</font>：

|参数名|作用|
|------|----|
|size|需要生成随机字符串的长度|

<font color=green>**代码实现**</font>：

```go
import (
	"fmt"
	"math/rand"
	"time"
)

var (
    // 全局的随机字符串取样样本
	char       = "zxcvbnmasdfghjklqwertyuiopZXCVBNMASDFGHJKLQWERTYUIOP1234567890"
	
    // 全局设置种子，避免在高并发下生成相同的随机数以及减小开销
    globalRand = rand.New(rand.NewSource(time.Now().UnixNano()))
)

// GenRandomString 生成指定长度的随机字符串
func GenRandomString(size uint8) string {
	name := ""
	for _ = range size {
		idx := RandomInt(0, 61)
		name += string(char[idx])
	}
	return name
}

type Integer interface {
	~int | ~int8 | ~int16 | ~int32 | ~int64 |
		~uint | ~uint8 | ~uint16 | ~uint32 | ~uint64
}

// RandomInt 返回指定范围内的随机整数 [min, max]
func RandomInt[T Integer](min, max T) T {
	// 生成随机数
	return T(globalRand.Intn(int(max-min+1)) + int(min))
}
```

# 3. ParseIntegerSliceToString

<font color=green>**方法签名**</font>： `ParseIntegerSliceToString[T Integer](data []T, split string) string`

<font color=green>**功能**</font>： 将整型切片解析为字符串. 例如字符串"1,2,3,4"， 通过`ParseIntegerSliceToString(str, ",")方法调用会解析成切片[1,2,3,4]`

<font color=green>**参数**</font>：

|参数名|作用|
|------|----|
|data|整型字切片列表|
|split|拼接符号|

<font color=green>**代码实现**</font>：

```go
import (
	"fmt"
	"strings"
)

type Integer interface {
	~int | ~int8 | ~int16 | ~int32 | ~int64 |
	~uint | ~uint8 | ~uint16 | ~uint32 | ~uint64
}

// ParseIntegerSliceToString 将整型切片解析为字符串
func ParseIntegerSliceToString[T Integer](data []T, split string) string {
	var result string
	for _, v := range data {
		result += fmt.Sprintf("%d%s", v, split)
	}
	result, _ = strings.CutSuffix(result, split)
	return result
}
```

# 4. ParseStringToIntegerSlice

<font color=green>**方法签名**</font>： `ParseStringToIntegerSlice[T Integer](data string, split string) (result []T, err error)`

<font color=green>**功能**</font>： 将字符串解析为整型切片, 例如对于整型切片[1,2,3,4], 通过连接符'-'生成的字符串为 "1-2-3-4"

<font color=green>**参数**</font>：

|参数名|作用|
|------|----|
|data|字符串|
|split|拼接符号|

<font color=green>**返回值**</font>：

<font color=green>**代码实现**</font>：

|参数名|作用|
|------|----|
|result|解析后的整型切片|

```go
import (
	"strconv"
	"strings"
)

type Integer interface {
	~int | ~int8 | ~int16 | ~int32 | ~int64 |
		~uint | ~uint8 | ~uint16 | ~uint32 | ~uint64
}

// ParseStringToIntegerSlice 将字符串解析为整型切片
func ParseStringToIntegerSlice[T Integer](data string, split string) ([]T, error) {
	slice := strings.Split(data, split)
	var result []T
	for _, v := range slice {
		num, err := strconv.ParseInt(v, 10, 64)
		if err != nil {
			return nil, err
		}
		result = append(result, T(num))
	}
	return result, nil
}
```

# 5 GetFirstLetter

<font color=green>**方法签名**</font>： `GetFirstLetter(s string) string`

<font color=green>**功能**</font>： 获取字符串的首字母

<font color=green>**参数**</font>：

|参数名|作用|
|------|----|
|string|字符串|

<font color=green>**返回值**</font>：

|参数名|作用|
|------|----|
|string|长度为1的字符串，为参数字符串的首字符|

<font color=green>**代码实现**</font>：

```go
import (
	"fmt"
	"github.com/mozillazg/go-pinyin"
	"strings"
	"unicode"
)

// GetFirstLetter 获取字符串的首字母
func GetFirstLetter(s string) string {
	if len(s) == 0 {
		return ""
	}

	r := []rune(s)[0]

	// 英文字母
	if unicode.IsLetter(r) && r <= unicode.MaxASCII {
		return strings.ToUpper(string(r))
	}

	// 汉字
	if unicode.Is(unicode.Han, r) {
		a := pinyin.Pinyin(string(r), pinyin.NewArgs())
		if len(a) > 0 && len(a[0]) > 0 {
			return strings.ToUpper(string(a[0][0][0]))
		}
	}

	// 其它符号
	return string(r)
}
```

# 6 RemoveAtIndex

<font color=green>**方法签名**</font>： `RemoveAtIndex[T any](s []T, index int) []T`

<font color=green>**功能**</font>： 删除切片中指定索引的元素

<font color=green>**参数**</font>：

|参数名|作用|
|------|----|
|s|切片|
|index|索引|

<font color=green>**返回值**</font>：

|参数名|作用|
|------|----|
|[]T|移除指定索引后的切片|

<font color=green>**代码实现**</font>：

```go
// RemoveAtIndex 删除切片中指定索引的元素
func RemoveAtIndex[T any](s []T, index int) []T {
	if index < 0 || index >= len(s) {
		// 索引越界，返回原切片（或可根据需要 panic 或返回 error）
		return s
	}
	return append(s[:index], s[index+1:]...)
}
```
